Game Programming Highlights

Placement system

The Placement System is a complex system that handles a series of functionality between multiple scripts. Its base functionality is to handle placing furniture of various sizes while compensating for said sizes and unique properties, handling rotation options and communicating with tiles that they can be placed on, all synced between players in servers.

Inventory system

The Inventory System goes hand & hand with the Placement System that handles data such as currency, collected furniture and the UI that goes along with it. The Inventory system allows users to interact with the tiles when furnishing their Cafe while syncing all data between players making sure said tile can be placed on by communicating with the placement system.

NPC Behaviour and Dialogue system

This system is for handling all NPC interactions and behaviour, This system uses a JSON to compile dialogue in our currently supported languages and feeds them into an interactions sequence that can have NPCs react accordingly. This also handles NPCs switching states along with their appropriate animations to compliment the main gameplay loop of our game.
